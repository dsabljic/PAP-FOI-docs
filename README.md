#  Upute za pokretanje

Nakon kloniranja git repozitorija treba kreirati virtual env

```bash
python3 -m venv env
```

Aktivacija evnvironmenta

```bash
source env/bin/activate
```

Deaktivacija

```bash
deactivate
```

Instaliranje potrebnih modula

```bash
pip3 install -r requirements.txt
```
